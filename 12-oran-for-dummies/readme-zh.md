# O-RAN for Dummies - 简易入门指南

## 概述
欢迎来到 O-RAN 世界！这是一个为完全新手设计的简易入门指南，用最简单的方式帮助您理解 O-RAN（开放无线接入网）技术。

## 为什么需要 O-RAN？

想象一下传统的手机网络就像一个封闭的城堡：
- 只有一个厂商建造所有的城墙和塔楼
- 如果想要改变或升级，必须找同一个厂商
- 成本高昂，选择有限

O-RAN 就像是开放的城市规划：
- 不同的建筑商可以建造不同的部分
- 可以自由选择最好的组件组合
- 更加灵活、经济、创新
