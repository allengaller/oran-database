## Financial Metrics Calculation

### Net Present Value (NPV)
```
NPV = Σ(CFt / (1+r)^t) - C0

Where:
CFt = Cash flow in year t
r = Discount rate (typically 8-12%)
C0 = Initial investment
t = Time period
```

### Internal Rate of Return (IRR)
```
IRR is the discount rate that makes NPV = 0
Can be solved using Excel IRR function or financial calculators
```

### Payback Period
```
Payback Period = Year when cumulative cash flow first becomes positive
```

## Decision Framework

### Investment Viability Criteria
- **High Viability**: NPV > 0 AND IRR > hurdle rate
- **Medium Viability**: NPV near 0 OR IRR slightly below hurdle rate
- **Low Viability**: NPV < 0 OR IRR significantly below hurdle rate
