## 财务指标计算

### 净现值 (NPV)
```
NPV = Σ(CFt / (1+r)^t) - C0

其中：
CFt = 第 t 年的现金流量
r = 折现率 (通常取 8-12%)
C0 = 初始投资
t = 时间周期
```

### 内部收益率 (IRR)
```
IRR 是使 NPV = 0 的折现率
可通过 Excel IRR 函数或财务计算器求解
```

### 投资回收期
```
回收期 = 累计现金流首次为正的年份
```

## 决策框架

### 投资可行性标准
- **高可行性**：NPV > 0 且 IRR > 门槛收益率
- **中等可行性**：NPV 接近 0 或 IRR 略低于门槛收益率
- **低可行性**：NPV < 0 或 IRR 明显低于门槛收益率

### 战略建议

#### 分阶段投资方法
1. **第一阶段**：试点部署 (目标规模的 20-30%)
   - 投资额：￥260万-540万
   - 时间线：6-12个月
   - 风险控制：受控环境测试

2. **第二阶段**：规模扩张 (剩余的 50-70%)
   - 投资额：￥650万-1300万
   - 时间线：12-18个月
   - 经验整合：纳入试点结果

3. **第三阶段**：全面部署 (剩余容量)
   - 投资额：￥390万-780万
   - 时间线：6-12个月
   - 优化配置：性能调优后的配置

#### 合作伙伴关系策略
- **技术合作伙伴**：与成熟厂商协作
- **服务提供商**：聘请有经验的系统集成商
- **研究机构**：利用学术界专业知识
- **产业联盟**：参与标准制定工作

## 结论与后续步骤

### 执行摘要
ROI 分析表明 O-RAN 部署具有强劲的财务可行性：
- 多种情景下均呈现正向 NPV
- 吸引人的 IRR 超越典型门槛收益率
- 合理的回收期在可接受范围内
- 实质性的风险调整后收益

### 实施路线图
1. **第1-2季度**：确定投资决策和合作伙伴关系
2. **第3-4季度**：执行第一阶段试点部署
3. **第2年**：扩展到第二阶段部署
