# O-RAN 基础

## 概述
本节涵盖O-RAN技术的基本概念和基础知识，包括其定义、愿景、历史演进、核心价值、关键概念和整体架构。作为云平台运维专业人员，这些知识将构成您O-RAN专业知识的基础。

## 关键主题

### 1. 架构概览
- **整体架构**：O-RAN联盟定义的标准架构图，组件关系
- **功能模块**：每个网络元素的功能定义和职责
- **网络元素关系**：组件之间的通信和交互机制
- **标准化目标**：接口标准化、功能标准化、部署标准化

### 2. 核心组件
- **无线电单元 (RU)**：射频处理功能，连接到天线
- **分布式单元 (DU)**：物理层和部分MAC层功能
- **集中式单元 (CU)**：RRC、PDCP等高层协议功能
- **RAN智能控制器 (RIC)**：Near-RT RIC和Non-RT RIC功能
- **服务管理与编排 (SMO)**：网络服务管理和编排功能

### 3. 接口标准
- **F1接口**：3GPP定义的CU-DU接口
- **开放前传接口 (O-FH)**：O-RAN联盟定义的标准化前传接口
- **E2接口**：RIC与CU/DU之间的接口
- **A1接口**：Non-RT RIC与Near-RT RIC之间的接口
- **O1接口**：SMO与其他网络元素之间的管理接口

### 4. 解耦选项
- **选项1-8**：8种解耦选项的详细说明
- **功能边界**：每个分割点处CU、DU、RU的具体功能边界
- **延迟要求**：不同解耦选项的性能要求
- **部署考虑因素**：技术、成本、运维方面的部署考虑

## 学习目标

1. **理解O-RAN参考架构**及其与传统RAN的区别
2. **识别关键网络元素**及其功能职责
3. **掌握接口标准**及其在多厂商环境中的重要性
4. **评估解耦选项**及其对网络设计的影响
