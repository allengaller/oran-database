## A

### A1 Interface / A1 接口 / Интерфейс A1
**English**: Policy management interface between Non-RT RIC and Near-RT RIC, based on RESTful architecture  
**中文**: 非实时RIC与近实时RIC之间的策略管理接口，基于RESTful架构  
**Русский**: Интерфейс управления политиками между не реального времени RIC и близкого реального времени RIC, основанный на архитектуре RESTful  

### AI/ML Framework / AI/ML 框架 / Фреймворк ИИ/МО
**English**: Artificial Intelligence/Machine Learning framework for O-RAN intelligent applications  
**中文**: 用于O-RAN智能应用的人工智能/机器学习框架  
**Русский**: Фреймворк искусственного интеллекта/машинного обучения для интеллектуальных приложений O-RAN  

### Antenna / 天线 / Антенна
**English**: Device that transmits and receives radio waves  
**中文**: 发射和接收无线电波的设备  
**Русский**: Устройство, передающее и принимающее радиоволны  

### API (Application Programming Interface) / 应用程序接口 / Интерфейс программирования приложений
**English**: Standardized interface for software components to communicate with each other  
**中文**: 软件组件间通信的标准化接口  
**Русский**: Стандартизированный интерфейс для взаимодействия программных компонентов друг с другом  

### Architecture / 架构 / Архитектура
**English**: Overall structure and organization of O-RAN system components  
**中文**: O-RAN系统组件的整体结构和组织  
**Русский**: Общая структура и организация компонентов системы O-RAN

**English**: Complete process from source to destination without intermediate processing  
**中文**: 从源到目的地的完整过程，无需中间处理  
**Русский**: Полный процесс от источника до получателя без промежуточной обработки

**中文**: 制定电信标准（包括O-RAN规范）的欧洲组织
