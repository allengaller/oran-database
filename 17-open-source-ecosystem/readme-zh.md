# O-RAN 开源生态系统

## 概述
本目录汇集 O-RAN 相关的开源项目、社区资源、代码示例和开发者工具，促进技术创新和生态发展。

## 主要开源项目

### 核心平台项目
- **ONF Aether** - 5G 边缘云平台
- **OpenCORD** - 云原生数据中心操作系统
- **ONOS** - SDN 控制器平台
- **CORD** - Central Office Re-architected as a Datacenter

### O-RAN 相关项目
- **O-RAN SC** - O-RAN 软件社区官方项目
- **RIC 相关开源** - 近实时和非实时 RIC 实现
- **CU/DU 开源实现** - O-CU 和 O-DU 开源版本
- **RU 接口开源** - O-RU 控制和用户平面实现

### 开发者工具
- **仿真器和测试工具** - 网络仿真、协议测试
- **SDK 和 API** - 开发工具包、接口文档
- **CI/CD 流水线** - 持续集成、自动化部署
- **监控和调试工具** - 性能分析、故障诊断
