# O-RAN Open Source Ecosystem

## Overview
This directory aggregates O-RAN related open source projects, community resources, code examples and developer tools to promote technological innovation and ecosystem development.

## Major Open Source Projects

### Core Platform Projects
- **ONF Aether** - 5G edge cloud platform
- **OpenCORD** - Cloud-native data center operating system
- **ONOS** - SDN controller platform
- **CORD** - Central Office Re-architected as a Datacenter

### O-RAN Related Projects
- **O-RAN SC** - O-RAN Software Community official projects
- **RIC Related Open Source** - Near-real-time and non-real-time RIC implementations
- **CU/DU Open Source** - O-CU and O-DU open source versions
- **RU Interface Open Source** - O-RU control and user plane implementations

### Developer Tools
- **Simulators and Testing Tools** - Network simulation, protocol testing
- **SDKs and APIs** - Development toolkits, interface documentation
- **CI/CD Pipelines** - Continuous integration, automated deployment
- **Monitoring and Debugging Tools** - Performance analysis, fault diagnosis
